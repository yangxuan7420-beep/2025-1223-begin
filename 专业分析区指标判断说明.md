# 专业分析区指标判断说明

本文档说明「专业分析区」中各个指标的判断原则，便于解读风险矩阵的三态输出（`True`=触发异常、`False`=未触发、`None`=缺失/不适用）。规则以 Logic 层现有实现为准，不在 View 层追加推断。

## 共性判定思路
- **数据来源与取值**：基础指标使用映射科目的首期有效数值；派生指标在此基础上做差或比值计算，不对原始数据做平滑或补全。
- **同比计算要求**：仅在年份连续的相邻两期存在有效数值、且分母不为 0 时计算同比；否则返回缺失状态并记录原因。
- **异常阈值**：同比异常阈值为 `|YoY| > 300%`（`YOY_ANOMALY_THRESHOLD=3.0`）；DSRI 异常阈值为 `DSRI > 1.4`。
- **缺失与不适用**：缺表、缺科目、分母为 0 或年份不连续时，返回 `None` 并带上具体提示，不会强行给出布尔结论。

## 基础指标（同比异常）
- **营业收入、经营现金流、货币资金、有息负债**：存在两期及以上且年份连续时计算同比；若某期缺值或基期为 0，则记为缺失原因，不触发判断；绝对同比超过 300% 视为异常。
- **利润（归母净利润优先，其次净利润）**：优先使用归母净利润，缺失时回退到净利润；未命中任何利润口径时标记为口径缺失；同比判定规则与其他基础指标一致。

## 派生指标
- **自由现金流（FCF）**：按「经营活动产生的现金流量净额 − 资本性支出」计算。任一科目缺失或未选定时记入缺失原因并返回 `None`；计算得到最新期值小于 0 时触发异常；不进行同比判定。
- **DSRI（应收账款 / 营业收入 的连续期比值之比）**：取最近两期的应收账款与营业收入；若任一期缺失或营业收入为 0，则返回 `None`；得到的 DSRI 值大于 1.4 时触发异常；不进行同比判定。

## 输出解读提示
- 风险矩阵按指标拆分「同比异常」与「派生异常」两类提示，不同指标可能只适用其中一类。
- 同比趋势与最新期值仅用于辅助展示，不改变异常判定的三态结果。
