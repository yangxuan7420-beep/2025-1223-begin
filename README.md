VisionFinance 是一个围绕财报文本与结构化解析的三层实验项目，补足 Wind 在二次分析与教学中的空白，而不是替代 Wind。

## 解决的问题
- Wind 导出财报常见痛点：导出格式多为 PDF 或粗粒度 Excel，二次整理耗时，课堂演示难以直接对齐结构化要点。
- 本项目聚焦“二次分析与结构化解释”：将已有财报数据解析为可复用的片段，方便教学、笔记和指标扫描。

## 系统架构
- Parser：负责读取并清洗从 Wind 导出的财报内容，输出统一的结构化中间层。
- Logic：在结构化数据上执行解释、摘要或特征归纳等逻辑，形成可消费的业务结论。
- View：将逻辑层产出以可视化或文本形式呈现，便于教学演示与快速浏览。

## 功能边界声明
- 不做原始数据查询，不负责直接访问或替代 Wind 的数据源。
- 不做实时行情，所有输入均基于既有导出数据。
- 不在 UI 中计算指标，计算与解释发生在 Logic 层，界面只展示结果。

## 适用人群
- 学生：用于数据分析或财务理解练习，便于对照结构化解释。
- 有基础的分析者：快速扫描关键指标或结论，加速复盘与讲解。
